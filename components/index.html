Button - 按钮

该组件是 Antd Button 的封装, 在其基础上增加了 visible 属性, 用于更方便的控制显示与隐藏。 该组件适用于比如按钮权限等场景。

阅读更多

LinkButton - 链接按钮

该组件是 Antd Link Button 的封装。与原来组件不同的是该组件去掉了 padding 和白色的底色，同时去掉了其固定高度，让其可以适应不同的背景底色，同时表现的更加紧凑。该组件适用于表格的操作列等空间不宽裕的地方。该组件同样有 visible 属性。

阅读更多

FormUpload - 表单上传

该组件是对 Antd Upload 组件的封装, 通过对 fileList 和 onChange 属性的接管, 让它表现为一个标准的 value - onChange 受控组件, 使其能够被直接用在 Form.Item 里.

FormUpload 组件的设计目标是希望在简单的全局配置过后, 使用 FormUpload 组件就像使用 Input 组件一样简单轻量. 上传成功的文件会被表单自动收集, 上传失败的文件会被自动过滤.

SettingProvider - 组件默认设置

为组件提供统一的默认设置，在外围包裹一次即可全局生效。
