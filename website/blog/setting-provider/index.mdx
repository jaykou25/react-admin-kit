# SettingProvider - 组件默认设置

为组件提供统一的默认设置，在外围包裹一次即可全局生效。

{/* truncate */}

## SchemaForm

比如可以统一配置 SchemaForm 默认的验证提示语。

<Previewer src="@@/SettingProvider/demos/schemaForm" />

## ModalForm

比如可以统一配置弹窗表单的默认布局。

<Previewer src="@@/SettingProvider/demos/modalForm" />

或者可以统一配置弹窗表单的默认宽度。

<Previewer src="@@/SettingProvider/demos/modalForm-width" />

## ProTable

比如可以统一配置分页选项。

<Previewer src="@@/SettingProvider/demos/protable-pagination" />

或者统一配置搜索区域。

<Previewer src="@@/SettingProvider/demos/protable-search" />

或者统一配置表格大小。

<Previewer src="@@/SettingProvider/demos/protable-size" />

## FormUpload

可以在这里统一设置 `action`, `header`, `children`, `responseToFileList` 等属性。
