# Internationalization (i18n)

Since RAK wraps Antd and Antd ProComponents, RAK's internationalization requires configuring Antd and Antd ProComponents in addition to RAK itself.

<Previewer src="@@/ProTable/demos/i18n"  />

RAK itself has minimal multilingual text, so other languages can be created by modifying the English version.
